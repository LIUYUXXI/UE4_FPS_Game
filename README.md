# UE4_FPS_Game
**UE4制作的第一人称射击游戏**
* 1、人物开火、瞄准、切换武器、装弹、子弹消耗逻辑的实现；
* 2、实现人物的冲刺、蹲起动作切换
* 3、实现了敌人的自动追踪视野内玩家功能；
* 4、实现了FPS游戏的基础UI面板，包括血量、子弹量和当前任务目标等;
* 5、实现了小地图
